# orgTree
An organizational framework selector.

> 基于Jquery编写的插件

调用方法：

	$('.form-control').orgTree({
        all: true,                //人物组织都开启
        area: ['620px','542px'],  //弹窗框宽高
        search: true              //开启搜索
	});


##### 说明：
	1、树结构添加json处理方法，欢迎借鉴
	2、数据结构可以根据属性进行按照字母顺序排序
